============================================================
PINTEREST POST PROMPT
============================================================

BRAND DETECTION: Follow _Brand_Detection_Rules.txt before generating.

Once the brand is confirmed, use the template below.

------------------------------------------------------------
TEMPLATE
------------------------------------------------------------

Create a Pinterest post for [BRAND NAME], [BRAND DESCRIPTION], for the blog post titled "[BLOG TITLE]". Generate a Title and Description. The title should be concise and descriptive with a subtitle. The description should be 1-2 sentences in B2B language targeting [TARGET AUDIENCE], followed by 4-5 relevant hashtags with the last hashtag always being [BRAND HASHTAG]. Do not use em dashes.

Blog URL: [INSERT BLOG URL]

------------------------------------------------------------
OUTPUT FORMAT (CSV)
------------------------------------------------------------

Sr. No. | Blog URL | Title | Description

------------------------------------------------------------
RULES
------------------------------------------------------------

- Title format: Main Topic: Descriptive Subtitle (e.g. Skills in Demand in Germany: Workforce Priorities)
- Description is 1-2 sentences max before hashtags
- 4-5 hashtags, last one is always the brand hashtag
- No em dashes anywhere
- Vary the exact phrasing of the description sentence(s) post to post, do not reuse the same description wording across a batch. Examples of the pattern (write new ones each time, do not copy these verbatim): "Built for teams that need results fast.", "A practical resource for the next planning cycle.", "Made for decision makers weighing this exact tradeoff."
- The identical description sentence reused across multiple pins in the same batch - if you are generating more than one Pinterest post in a session, no two may share the same description wording
- Output saved to output/Pinterest/
